◆ impf_mat_transmajor_inplace()

void impf_mat_transmajor_inplace ( impf_t_matrix *  mat)

Change the major of a matrix (just for testing).

In-place matrix transpose [1]. Generally, the inclass matrix transpose is slower than using auxiliary memory.

Also see: https://en.wikipedia.org/wiki/In-place_matrix_transposition

Example: file eg_dts_impf_mat_transmajor_inplace.c

#include <stdio.h>
#include <impf/impf.h>
int
main(void)
{
/* Create a matrix */
double data[] = {1., 2., 3., 4., 5., 6.};
impf_t_matrix mat = {data, 2, 3, IMPF_MAT_ROW_MAJOR};
/* Change the major of a matrix (using inplace version) */
impf_mat_transmajor_inplace(&mat);
/* Print the matrix */
FILE * out = fopen("eg_dts_impf_mat_transmajor_inplace.out", "w");
if(NULL == out) return -1;
impf_fprint_mat_sci(out, &mat, 100, 1);
fclose(out);
return 0;
}
