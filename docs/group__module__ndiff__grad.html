Gradient

Description

The gradient vector of \(f(x): \mathbb{R}^n\to\mathbb{R}\) is a \(n\)-dimensional double array.

The derivatives are calculated using a 5-points method: Given a step length \(h\), the derivative of real function \(f:\mathbb{R}\to\mathbb{R}\) at \(x\) is calculated by

\[ f'(x) \approx \frac{f(x-2h)-f(x+2h)+8[f(x+h)-f(x-h)]}{h} \]

The error is of the degree \(\mathcal{O}(h^5)\). The step length \(h\) is chosen as \(\sqrt{\epsilon}\), where \(\epsilon\) is the double epsilon defined in <float.h>.
