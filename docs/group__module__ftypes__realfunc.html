Real Functions

Description

Real functions refers to those user-defined functions with real inputs and return that IMPF recognizes.

Note that IMPF is a "real" numerical package which focus on double float values. Real functions have double inputs and double outputs. The input could be a double value (scalar), several double values, a double array (pointer), a double vector (pointer), or several double arrays/vectors. The return could be a double scalar or a double array/vector.

The real functions of 1-variant, 2-variant, 3-variant and n-variants that IMPF recognizes are defined here.
