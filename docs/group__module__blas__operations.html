Vector & Matrix Basic Operations

Description

There are three basic (row) operations for matrices:

  1. Swapping two rows;
  2. Scaling a row by a scalar;
  3. Plusing on a row another scaled row.

Methods of vector based and matrix based versions of basic (row) operations are defined below:
