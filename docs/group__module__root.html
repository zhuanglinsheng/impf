Gauss-Newton Algorithms

This method search the unconstraint root of a function \(f:\mathbb{R}^m\to\mathbb{R}^n\).

When \(m = n\), the algorithm is degerated to a classical newton's method. We update the guesses \(x\) by

\[ x_{t+1} = x_t - J^{-1}(x_t)f(x_t) \]

When \(m > n\), it is natural to use \(J^+ = (J^TJ)^{-1}J^T\) as a generalized Jacobian, and find a root in the nonlinear-least square sense.
